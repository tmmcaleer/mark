## Plugin Signing Process
When ready for deployment, the avpi plugin file must be submitted to Avid for signing. Signing is required in order for an avpi plugin to be deployed onto a users Avid Media Composer system for production use. 

Before submitting the avpi plugin file for signing and to avoid delays, please ensure that:
- avid-manifest.json is correctly completed
- in avid-manifest.json that "companyPrefix" is changed from "avid" and replaced with your own preferred company prefix
- avpi file contains **contents** of the project directory rather than the entire directory including project folder

### Signing
- Before signing the avpi is unzipped and inspected
- At Avid's discretion, minor issues may be corrected, otherwise the avpi plugin will be rejected and submitter asked to fix issues before submitting again.
- The avpi plugin package will be signed
