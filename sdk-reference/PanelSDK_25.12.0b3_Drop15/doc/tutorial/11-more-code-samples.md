# More code samples

The PanelSDK package contains several sample projects which demonstrate how to use the API. Most of them are Nodejs projects. You need to install Nodejs from https://nodejs.org/.

You can find more samples in PanelSDK/samples. We recommend you to install the `basic-api` sample which is included at `PanelSDK/samples/basic-api`. This particular sample is “self-contained”, which  means you don’t have to create a server to provide the contents for the web page. With self-contained plugin, all the resources needed are included in the bundle. All you need is  to build and install it.
