# Media Composer Panel SDK Documentation

The Media Composer Panel SDK provides APIs to access various Media Composer features such as Bin Metadata and Import/Export. To use the Panel SDK APIs, the plugin will need to provide an HTML UI that will be displayed in a Media Composer window after the user has opened a project. The plugin's manifest file specifies a menu pick name that the user can use to bring up the plugin's UI. APIs are called via gRPC calls. The Panel SDK kit provides documentation for APIs, notifications, along with samples and debugging tips.
